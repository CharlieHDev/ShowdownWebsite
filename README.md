# 🖥️ MC Showdown Website
 
The repository for the MC Showdown website — holding all information about the event with event and player statistics easily accessible for the community.

🌐 Visit the website at: [mcshowdown.org](https://mcshowdown.org)
 
> ## What is Showdown?
> Showdown is a 32-player Minecraft Tournament with custom-coded minigames.
> See the [Showdown Repository](https://github.com/CharlieHDev/Showdown2.git) for the game code.
